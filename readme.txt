	ユーティリティライトは先に製作で本体を作る必要があります。
	効果のある家具なので、それなりの材料とスキルが必要になるようになっています。
	(必要スキル：電子工学6、 原子力電池など)
	その他の家具は、今は基本的に効果はない物が多いです。
	
	また、スロットマシンは10ドル入れて遊ぶ事が可能です。役が揃えば
	お金が支払われたりします。
